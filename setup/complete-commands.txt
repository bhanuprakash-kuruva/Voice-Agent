Below is the **complete, linear command sequence** starting from **cluster creation → platform install → AI stack → observability → vmagent**, using the **exact YAML stack you are using** (including the `promtail.yaml` you pasted).

This is written so someone can **recreate your entire system from scratch without debugging**.

# TERMINAL 1

# Create Kubernetes Cluster + Platform

---

# 1. Check Existing Clusters

```bash
k3d cluster list
```

Delete old cluster if needed:

```bash
k3d cluster delete kvoice-cluster
```

---

# 2. Create the Kubernetes Cluster

```bash
k3d cluster create kvoice-cluster \
--agents 1 \
--image rancher/k3s:v1.31.5-k3s1 \
-p "11434:30080@agent:0" \
-p "30300:30300@agent:0" \
-p "30750:30750@agent:0" \
-p "30100:30100@agent:0" \
-p "30200:30200@agent:0" \
-p "30417:30417@agent:0" \
-v /usr/share/ollama/.ollama/models:/models@all
```

### Port mapping

| Port  | Service         |
| ----- | --------------- |
| 11434 | Ollama          |
| 30300 | Grafana         |
| 30750 | VictoriaMetrics |
| 30100 | Loki            |
| 30200 | Tempo           |
| 30417 | Tempo OTLP      |

---

---

# 8. Configure Knative to Use Kourier

```bash
kubectl patch configmap/config-network \
-n knative-serving \
--type merge \
-p '{"data":{"ingress-class":"kourier.ingress.networking.knative.dev"}}'
```

# WINDOWS

# Start vmagent

Download:

```
https://github.com/VictoriaMetrics/VictoriaMetrics/releases
```

Download:

```
vmutils-windows-amd64-v1.137.0.zip
```

Extract.

---

# Create vmagent.yml

```yaml
global:
  scrape_interval: 5s

scrape_configs:

  - job_name: voice_rag_pipeline
    metrics_path: /metrics
    static_configs:
      - targets:
        - 127.0.0.1:8000

  - job_name: kube_state_metrics
    metrics_path: /metrics
    static_configs:
      - targets:
        - <WSL-IP>:8081
```

Find WSL IP:

```bash
ip addr | grep eth0
```

Example:

```
172.20.236.51
```

---

# Start vmagent

In PowerShell:

```powershell
.\vmagent-windows-amd64-prod.exe "-promscrape.config=vmagent.yml" "-remoteWrite.url=http://127.0.0.1:30750/api/v1/write"
```
